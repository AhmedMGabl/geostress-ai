# Research References

## ML for Fracture Characterization in Oil Reservoirs

- [Data-driven fracture intensity prediction using ML for presalt carbonates](https://www.sciencedirect.com/science/article/pii/S2666759225000253) - XGBoost/Random Forest for fracture intensity from seismic attributes
- [ML-Based Seismic Subsurface Characterization: State of the Art (2025)](https://agupubs.onlinelibrary.wiley.com/doi/full/10.1029/2025JH000846) - Comprehensive review of DL in seismic characterization
- [ML for Fracture Parameter Estimation from Seismic Data (SPE)](https://onepetro.org/SPERPTC/proceedings/20RPTC/20RPTC/D023S007R007/450195) - SVM, RF, XGBoost, BPNN for fracture prediction
- [Unsupervised ML for fault/fracture network interpretation](https://link.springer.com/article/10.1007/s40948-023-00646-9) - Vector quantization neural networks
- [Optimized ML models for natural fractures prediction](https://www.sciencedirect.com/science/article/abs/pii/S001623612201794X) - Using conventional well logs

## AI Geostress Inversion

- [Combining ML and physics for real-time geostress determination](https://www.sciencedirect.com/science/article/pii/S3050619024000041) - ML-Physics surrogate ANN model
- [Physics-informed neural network for tectonic stresses](https://www.nature.com/articles/s41598-023-50759-0) - PINNs for stress field estimation
- [StressNet: Deep learning for stress prediction with fracture propagation](https://www.nature.com/articles/s41529-021-00151-y) - TI-CNN + Bi-LSTM
- [Accurate measurement and prediction of in-situ rock stress](https://www.nature.com/articles/s41598-024-64030-7) - GM-BPNN model
- [Geomechanical paleostress inversion using fracture data](https://www.sciencedirect.com/science/article/abs/pii/S0191814116300839)
